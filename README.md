# Uni2Go

Proyecto hecho con **Vite**.

---

## Instalación de dependencias

1. Abre la terminal y entra en la carpeta `frontend`:

   cd frontend

2. Instala todas las dependencias:

   npm install

---

## Ejecución del proyecto

### Modo desarrollo

    npm run dev

Se abrirá normalmente en http://localhost:5173.

### Build y previsualización

    npm run build
    npm run preview

Se genera la carpeta dist y se puede previsualizar localmente.

---

## Conexión con la base de datos

Crea un archivo `.env` dentro de `frontend` con las variables necesarias.  
Asegúrate de que `.env` esté en `.gitignore`.
